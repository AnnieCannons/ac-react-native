Maps and Props

Components are the same

In React

import React from 'react';
class HelloWorldApp extends React.Component{
  render(){
    return (
      <h1>Hello World</h1>
    )
  }
}

becomes....

import React, { Component } from 'react';
import {Text} from 'react-native';
class HelloWorldApp extends Component{
  render(){
    return (
      <Text>Hello World</Text>
    )
  }
}

CSS

Stylesheet

All components have a style prop that accepts a JavaScript object.

You can use Stylesheet.create() to create a more compoent style sheet object that you can reference in your components.

Example


import React, { Component } from 'react';
import { StyleSheet, Text, View } from 'react-native';

class LotsOfStyles extends Component {
  render() {
    return (
      <View>
       <Text style={styles.red}>just red</Text>
       <Text style={styles.bigblue}>just bigblue</Text> 
       <Text style={[styles.bigblue, styles.red]}>bigblue, then red</Text>
       <Text style={[styles.red, styles.bigblue]}>red, then bigblue</Text>
      </View>
    );
  }
}

Example

	
const styles = StyleSheet.create({
  bigblue: {
    color: 'blue',
    fontWeight: 'bold',
    fontSize: 30,
  },
  red: {
    color: 'red',
  }
});

Flexbox

Mobile viewports have fixed dimensions

React Native allows you to use flexbox to layout components, just like on the web.

BUT the defaults are different, ie flexDirection is defaulted to column and alignItems is defaulted to stretch instead of flex-start.

Creating a React Native App

  1. Make sure you have Node installed
  2. Make sure you have Watchman as well: brew install watchman
  3. Install the React Native CLI npm install -g react-native-cli
  4. Xcode: The easiest way to install Xcode is via the Mac App Store.
  5. Open Xcode, then choose "Preferences..." from the Xcode menu. Go to the Locations panel and install the dev tools by selecting the most recent version in the Command Line Tools dropdown.

Creating a new application

Change directories into the folder you want and enter the following command:


react-native init name-of-project

Install your dependecies

npm install --save name-of-dependency

Launch the app

cd AwesomeProject
react-native run-ios
